# Task: シナリオタブと比較ビューの実装

## ⚠️ Engine API 修正ノート（2026-04-21 追記）

本タスクファイルの「Technical Details」コード例は engine API 名に誤りがあります。**実装時は以下の実名を使用してください**:

| Task仕様（誤） | 実装（正） |
|---|---|
| `calc_single_cost(item, cond)` | `calc_single_item(item, cond)` |
| `calc_gift_cost(gift, cond, item_results)` | `calc_gift_set(gift, items, item_results, cond)` ※ `items`/`item_results` は **`dict[str, ...]`** |
| `.cost_jpy` | `.jpy_cost` |
| `.margin_pct` (GiftSetResult) | `.gross_profit_rate` |

**比較ビューでの再計算パターン**（正しい例）:
```python
# items は dict[str, SingleItem] としてメインセッションから渡される想定
left_item_results = {
    no: calc_single_item(item, left_s.condition)
    for no, item in items.items()
}
left_gift_results = [
    calc_gift_set(g, items, left_item_results, left_s.condition)
    for g in gifts
]
```

**シグネチャ訂正**: `render_scenarios` の第2引数 `items` は `dict[str, SingleItem]`、`item_results` は `dict[str, SingleItemResult]` として受け取ること（既存 `section_items.py` / `section_gift.py` と整合）。

## Description

シナリオ一覧・CRUD・2件横並び比較を提供する「シナリオ」タブを新規実装する。PRD の US-3（社長が複数シナリオを横並び比較）と US-4（営業がシナリオを名前付きで保存）を満たす中核タスク。

`proto/storage/scenario_repo.py`（タスク002）を使ってSQLite永続化、`streamlit-aggrid`（タスク004）で一覧表示、`kpi_cards`（タスク003）で比較ビューのKPI差分表示を行う。

**このエピックで最も複雑なタスク**。受入基準を細かく分けて段階的に実装する。

### 画面構成

#### 1. 上部アクションバー
- [ ] 「➕ 現在の条件を新規保存」ボタン — 名前入力 → `repo.save_scenario()` → トースト通知
- [ ] 同名がある場合: エラー表示「この名前は既に使われています」
- [ ] 保存後、一覧に反映（`st.rerun()` または session_state 更新）

#### 2. シナリオ一覧（aggrid）
- [ ] 列: チェックボックス（選択）・名前・作成日時・更新日時
- [ ] 選択モード: multiple
- [ ] 行ソート・フィルタ有効
- [ ] シナリオが 0 件の場合: `st.info("保存されたシナリオはありません。現在の条件を保存してください。")`

#### 3. アクションボタン群（一覧の下）
- [ ] 「📂 読み込み」ボタン — 1件選択時のみ活性化、サイドバーの session_state を更新 → `st.rerun()`
- [ ] 「📋 複製」ボタン — 1件選択時のみ活性化、新名前入力（デフォルト `{元の名前}_コピー`）
- [ ] 「✏️ リネーム」ボタン — 1件選択時のみ活性化、新名前入力
- [ ] 「🗑️ 削除」ボタン — 1件選択時のみ活性化、`st.confirm` 相当（`st.modal` または確認チェックボックス）
- [ ] 「📊 比較」ボタン — **2件選択時のみ活性化**、下部に比較ビュー展開

#### 4. 比較ビュー（2件選択 → 「比較」クリック時に展開）
- [ ] 左右2カラム構成
- [ ] 各カラムのヘッダ: シナリオ名
- [ ] 各カラムに以下を表示:
  - [ ] 条件サマリ（コンテナ / USD / マージン）
  - [ ] KPIカード3枚（平均原価・平均上代・平均粗利率）
  - [ ] 品目別原価テーブル（aggrid、主要列のみ）
  - [ ] ギフトセット原価テーブル（aggrid、主要列のみ）
- [ ] 差分表示:
  - [ ] KPIカードの delta 部分に「+¥123（左比）」を表示（右カラム側）
  - [ ] 改善は緑（`delta_color="normal"`）、悪化は赤

### データフロー
- [ ] 「比較」選択時: 2シナリオの `condition` を取り出し、`calc_single_cost` と `calc_gift_cost` で再計算（既存エンジン使用）
- [ ] 再計算結果を使って KPI と テーブルを描画
- [ ] 現在のサイドバー条件（`cond`）は比較には使わない（保存済みシナリオ同士の比較）

### 復元フロー（重要）
- [ ] 「読み込み」時、`condition` の全フィールドを session_state の対応キーに書き込み、サイドバーの次回描画時に復元されるようにする
- [ ] サイドバーの widget key 命名規約（`{selected}_{field_name}` 形式、既存コード参照）に合わせること

## Technical Details

### `proto/ui/scenarios.py` 骨格

```python
"""シナリオ管理タブ — 一覧・CRUD・横並び比較."""
from __future__ import annotations

from dataclasses import replace
from datetime import datetime, timezone

import pandas as pd
import streamlit as st

from proto.engine.calc_single import calc_single_cost
from proto.engine.calc_gift import calc_gift_cost
from proto.engine.models import (
    GiftResult,
    GiftSet,
    ImportCondition,
    SingleItem,
    SingleItemResult,
)
from proto.storage.scenario_repo import (
    ScenarioNameConflictError,
    ScenarioNotFoundError,
    ScenarioRepository,
)
from proto.ui.components.aggrid_table import (
    create_aggrid,
    currency_column,
    percent_column,
    text_column,
)
from proto.ui.components.kpi_cards import (
    KPICardData,
    format_currency,
    format_delta,
    format_percentage,
    render_kpi_row,
)


def _render_save_form(cond: ImportCondition, repo: ScenarioRepository) -> None:
    with st.expander("➕ 現在の条件を新規保存", expanded=False):
        name = st.text_input(
            "シナリオ名",
            value=f"{cond.name}_{datetime.now(timezone.utc).strftime('%Y%m%d_%H%M')}",
            key="new_scenario_name",
        )
        if st.button("保存", key="btn_save_scenario"):
            if not name.strip():
                st.error("名前を入力してください")
                return
            try:
                sid = repo.save_scenario(name.strip(), cond)
                st.success(f"✅ シナリオ '{name}' を保存しました (id={sid})")
                st.rerun()
            except ScenarioNameConflictError:
                st.error(f"この名前は既に使われています: {name}")


def _render_list_table(repo: ScenarioRepository) -> list[dict]:
    metas = repo.list_scenarios()
    if not metas:
        st.info("保存されたシナリオはありません。現在の条件を保存してください。")
        return []

    df = pd.DataFrame(
        [
            {
                "id": m.id,
                "名前": m.name,
                "作成日時": m.created_at[:19].replace("T", " "),
                "更新日時": m.updated_at[:19].replace("T", " "),
            }
            for m in metas
        ]
    )
    result = create_aggrid(
        df,
        column_defs=[
            text_column("名前", "名前", width=240),
            text_column("作成日時", "作成日時"),
            text_column("更新日時", "更新日時"),
        ],
        selection="multiple",
        height=300,
    )
    return result.selected_rows or []


def _render_action_buttons(
    selected: list[dict],
    repo: ScenarioRepository,
    cond: ImportCondition,
) -> None:
    col1, col2, col3, col4, col5 = st.columns(5)
    single = len(selected) == 1
    double = len(selected) == 2

    with col1:
        if st.button("📂 読み込み", disabled=not single, key="btn_load"):
            _load_scenario_into_sidebar(repo.get_scenario(selected[0]["id"]).condition)
            st.rerun()

    with col2:
        if st.button("📋 複製", disabled=not single, key="btn_duplicate"):
            _handle_duplicate(repo, selected[0]["id"], selected[0]["名前"])

    with col3:
        if st.button("✏️ リネーム", disabled=not single, key="btn_rename"):
            _handle_rename(repo, selected[0]["id"], selected[0]["名前"])

    with col4:
        if st.button("🗑️ 削除", disabled=not single, key="btn_delete"):
            _handle_delete(repo, selected[0]["id"], selected[0]["名前"])

    with col5:
        if st.button("📊 比較", disabled=not double, key="btn_compare"):
            st.session_state["compare_ids"] = [s["id"] for s in selected]


def _load_scenario_into_sidebar(loaded_cond: ImportCondition) -> None:
    """サイドバーの widget key に対応する session_state を更新して復元する."""
    # widget key 命名規約: {selected}_{field}
    # ※サイドバー実装（タスク008）と整合を取ること
    kp = loaded_cond.name.replace(" ", "_").replace("/", "_")
    st.session_state[f"{kp}_internal_rate"] = loaded_cond.internal_rate
    st.session_state[f"{kp}_current_rate"] = loaded_cond.current_rate
    st.session_state[f"{kp}_margin_pct"] = loaded_cond.margin_pct
    # ... (全フィールド対応、サイドバー実装に合わせて網羅)
    st.toast(f"シナリオ '{loaded_cond.name}' を読み込みました")


def _render_compare_view(
    ids: list[int],
    repo: ScenarioRepository,
    items: list[SingleItem],
    gifts: list[GiftSet],
) -> None:
    left_s = repo.get_scenario(ids[0])
    right_s = repo.get_scenario(ids[1])

    st.subheader(f"📊 比較: {left_s.name}  vs  {right_s.name}")

    # 各シナリオで再計算
    left_item_results = [calc_single_cost(it, left_s.condition) for it in items]
    right_item_results = [calc_single_cost(it, right_s.condition) for it in items]
    left_gift_results = [calc_gift_cost(g, left_s.condition, left_item_results) for g in gifts]
    right_gift_results = [calc_gift_cost(g, right_s.condition, right_item_results) for g in gifts]

    col_l, col_r = st.columns(2)
    with col_l:
        _render_scenario_column(left_s, left_item_results, left_gift_results, baseline=None)
    with col_r:
        _render_scenario_column(right_s, right_item_results, right_gift_results, baseline=left_item_results)


def _render_scenario_column(
    scenario,
    item_results: list[SingleItemResult],
    gift_results: list[GiftResult],
    *,
    baseline: list[SingleItemResult] | None,
) -> None:
    st.markdown(f"**{scenario.name}**")
    st.caption(
        f"🚢 {scenario.condition.name} | 💱 USD={scenario.condition.internal_rate:.0f}"
        f" | 🎯 {scenario.condition.margin_pct:.0f}%"
    )

    avg_cost = sum(r.cost_jpy for r in item_results) / len(item_results) if item_results else 0
    baseline_cost = (
        sum(r.cost_jpy for r in baseline) / len(baseline) if baseline else None
    )
    delta = format_delta(avg_cost, baseline_cost, is_currency=True) if baseline_cost is not None else None

    cards = [
        KPICardData(
            label="平均原価",
            value=format_currency(avg_cost),
            delta=delta,
            delta_color="inverse",  # 原価は低いほど良い
        ),
        # ... 平均上代・平均粗利率も同様
    ]
    render_kpi_row(cards)
    # 品目別原価テーブル・ギフトテーブル（aggrid）


def render_scenarios(
    cond: ImportCondition,
    items: list[SingleItem],
    item_results: list[SingleItemResult],
    gifts: list[GiftSet],
    gift_results: list[GiftResult],
    repo: ScenarioRepository,
) -> None:
    _render_save_form(cond, repo)
    selected = _render_list_table(repo)
    _render_action_buttons(selected, repo, cond)

    compare_ids = st.session_state.get("compare_ids")
    if compare_ids:
        st.divider()
        _render_compare_view(compare_ids, repo, items, gifts)
```
