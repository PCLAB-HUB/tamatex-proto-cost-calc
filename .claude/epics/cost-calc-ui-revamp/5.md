# Task: aggridテーブルファクトリの実装

## Description

`streamlit-aggrid` を使った高機能テーブル描画を、プロジェクト全体で統一された作法で呼び出せるファクトリ関数を実装する。通貨・パーセンテージ列のフォーマット、選択行の取得、ソート・フィルタの有効化を共通化する。

ダッシュボード・シナリオ一覧・既存の単品/ギフト/比較タブで全て使い回すため、**このファクトリの API 設計がUI実装全体の工数を左右する**。

### 新規ファイル

### API

- [ ] ソート有効（デフォルトで全列）

- [ ] ヘッダのチェックボックス選択（`selection="multiple"` 時）
- [ ] 戻り値から `.selected_rows` で選択行を取得可能

## Technical Details

### `proto/ui/components/aggrid_table.py` 設計

```python
"""streamlit-aggrid ファクトリ — 通貨・パーセント列の共通化."""
from __future__ import annotations

from typing import Any, Literal

import pandas as pd
import streamlit as st
from st_aggrid import AgGrid, GridOptionsBuilder, ColumnsAutoSizeMode
from st_aggrid.shared import GridUpdateMode, DataReturnMode

_YEN_FORMATTER = """
function(params) {
  if (params.value == null) return '';
  return '¥' + Number(params.value).toLocaleString('ja-JP', {maximumFractionDigits: 0});
}
"""

_PERCENT_FORMATTER_TEMPLATE = """
function(params) {
  if (params.value == null) return '';
  return Number(params.value).toFixed({decimals}) + '%';
}
"""


def currency_column(field: str, header: str, *, width: int | None = None) -> dict:
    col: dict[str, Any] = {
        "field": field,
        "headerName": header,
        "type": ["numericColumn", "rightAligned"],
        "valueFormatter": _YEN_FORMATTER,
        "sortable": True,
        "filter": "agNumberColumnFilter",
    }
    if width:
        col["width"] = width
    return col


def percent_column(field: str, header: str, *, decimals: int = 1, width: int | None = None) -> dict:
    col: dict[str, Any] = {
        "field": field,
        "headerName": header,
        "type": ["numericColumn", "rightAligned"],
        "valueFormatter": _PERCENT_FORMATTER_TEMPLATE.format(decimals=decimals),
        "sortable": True,
        "filter": "agNumberColumnFilter",
    }
    if width:
        col["width"] = width
    return col


def text_column(field: str, header: str, *, width: int | None = None) -> dict:
    col: dict[str, Any] = {
        "field": field,
        "headerName": header,
        "sortable": True,
        "filter": "agTextColumnFilter",
    }
    if width:
        col["width"] = width
    return col


def number_column(field: str, header: str, *, decimals: int = 0, width: int | None = None) -> dict:
    col: dict[str, Any] = {
        "field": field,
        "headerName": header,
        "type": ["numericColumn", "rightAligned"],
        "valueFormatter": f"function(params) {{ return Number(params.value).toFixed({decimals}); }}",
        "sortable": True,
    }
    if width:
        col["width"] = width
    return col


def create_aggrid(
    df: pd.DataFrame,
    column_defs: list[dict],
    *,
    selection: Literal["none", "single", "multiple"] = "none",
    height: int = 400,
    theme: str = "streamlit",
):
    """streamlit-aggrid でテーブルを描画し、戻り値から selected_rows 等を取得する."""
    gb = GridOptionsBuilder.from_dataframe(df)
    for col_def in column_defs:
        gb.configure_column(**col_def)
    gb.configure_default_column(
        resizable=True,
        filter=True,
        sortable=True,
    )
    if selection != "none":
        gb.configure_selection(
            selection_mode="single" if selection == "single" else "multiple",
            use_checkbox=True if selection == "multiple" else False,
        )
    grid_options = gb.build()

    return AgGrid(
        df,
        gridOptions=grid_options,
        height=height,
        theme=theme,
        columns_auto_size_mode=ColumnsAutoSizeMode.FIT_CONTENTS,
        update_mode=GridUpdateMode.SELECTION_CHANGED,
        data_return_mode=DataReturnMode.AS_INPUT_AND_FILTERED,
        allow_unsafe_jscode=True,  # valueFormatter に JS を使うため必須
    )
```

### 使用例（docstring 用）

```python
import pandas as pd
from proto.ui.components.aggrid_table import (
    create_aggrid,
    text_column,
    currency_column,
    percent_column,
)

df = pd.DataFrame({
    "品名": ["タオルA", "タオルB", "タオルC"],
    "原価": [1000, 1500, 2000],
    "粗利率": [45.5, 52.3, 48.9],
})

columns = [
    text_column("品名", "品名", width=200),
    currency_column("原価", "原価"),
    percent_column("粗利率", "粗利率"),
]

result = create_aggrid(df, columns, selection="multiple", height=300)
selected = result.selected_rows  # 選択された行の DataFrame
```

### 注意点
- `allow_unsafe_jscode=True` は aggrid の JavaScript formatter を使うため必須。Streamlit Cloud でも動作する
- `ColumnsAutoSizeMode.FIT_CONTENTS` で列幅を内容に合わせる（明示指定 width があればそれを優先）
- `theme="streamlit"` で Streamlit ネイティブのスタイルと統一
- バーチャートセル（粗利率）は初期版では valueFormatter のみ。複雑な cellRenderer は Phase 2 以降で検討
