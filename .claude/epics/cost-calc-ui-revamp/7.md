# Task: ダッシュボードタブの実装

## ⚠️ Engine API 修正ノート（2026-04-21 追記）

本タスクファイルの「Technical Details」に含まれるコード例は engine API 名に誤りがあります。**実装時は以下の実名を使用してください**（Phase 2 の #6 で発見、既存 `proto/ui/section_items.py` / `section_gift.py` も参照）:

| Task仕様（誤） | 実装（正） |
|---|---|
| `calc_single_cost(item, cond)` | `calc_single_item(item, cond)` |
| `calc_gift_cost(gift, cond, item_results)` | `calc_gift_set(gift, items, item_results, cond)` ※ `items` と `item_results` は **`dict[str, ...]`（品番キー）** |
| `.cost_jpy` | `.jpy_cost` (SingleItemResult) |
| `.margin_pct` (GiftSetResult) | `.gross_profit_rate` (GiftSetResult) |
| `MOCK_ITEMS` | `ALL_ITEMS` (`from proto.data.mock_items import ALL_ITEMS`) |
| `MOCK_GIFTS` | `ALL_GIFTS` (`from proto.data.mock_gifts import ALL_GIFTS`) |

**データ型の訂正**: `items` と `item_results` は `dict[str, SingleItem]` / `dict[str, SingleItemResult]` です。ループは `for item in items.values():` 等。既存の `section_items.py` / `section_gift.py` に従うこと。

**粗利率の単位**: `GiftSetResult.gross_profit_rate` は **0〜1 の小数**（例: 0.506）で返ります。% 表示するには `* 100` するか、`:.1%` 書式を使ってください。

## Description

アプリ起動時のトップビューとなる「ダッシュボード」タブを新規実装する。営業・役員・社長が開いた瞬間に主要KPI・為替感度・品目別原価ランキングを一目で把握できるようにする。

PRD の US-1（営業が商談中に即確認）と US-2（役員の為替感度把握）を満たす中核タスク。既存計算結果（単品・ギフト・サマリ）を受け取って表示するだけなので、**計算ロジックは一切追加しない**。

## Technical Details

### `proto/ui/dashboard.py` 設計

```python
"""ダッシュボードタブ — KPI・為替感度・ランキング表示."""
from __future__ import annotations

import plotly.graph_objects as go
import streamlit as st

from proto.engine.models import (
    GiftResult,
    GiftSet,
    ImportCondition,
    SingleItem,
    SingleItemResult,
    SummaryResult,
)
from proto.ui.components.kpi_cards import (
    KPICardData,
    format_currency,
    format_delta,
    format_percentage,
    render_kpi_row,
)
from proto.ui.components.sensitivity_chart import render_fx_sensitivity_chart


def _render_header(cond: ImportCondition) -> None:
    st.markdown(
        f"#### 🚢 {cond.name} | 💱 USD={cond.internal_rate:.0f} | 🎯 マージン目標 {cond.margin_pct:.0f}%"
    )
    st.divider()


def _render_kpi_cards(
    item_results: list[SingleItemResult],
    summary: SummaryResult,
    target_margin_pct: float,
) -> None:
    if not item_results:
        st.info("品目データがありません")
        return

    avg_cost = sum(r.cost_jpy for r in item_results) / len(item_results)
    # 上代 = 原価 / (1 - マージン%)
    avg_retail = avg_cost / (1 - target_margin_pct / 100)
    # 実効粗利率 = (上代 - 原価) / 上代 * 100
    actual_margin = (avg_retail - avg_cost) / avg_retail * 100 if avg_retail > 0 else 0

    cards = [
        KPICardData(
            label="平均原価",
            value=format_currency(avg_cost),
            sublabel=f"{len(item_results)}品目の平均",
        ),
        KPICardData(
            label="平均上代",
            value=format_currency(avg_retail),
            sublabel=f"（マージン{target_margin_pct:.0f}%想定）",
        ),
        KPICardData(
            label="平均粗利率",
            value=format_percentage(actual_margin),
            sublabel=f"目標 {target_margin_pct:.0f}% との差: {format_delta(actual_margin, target_margin_pct)}",
            delta=format_delta(actual_margin, target_margin_pct),
            delta_color="normal",
        ),
    ]
    render_kpi_row(cards)


def _render_item_ranking(
    items: list[SingleItem],
    item_results: list[SingleItemResult],
) -> None:
    st.subheader("📊 品目別原価ランキング")
    if not item_results:
        st.info("品目データがありません")
        return

    pairs = sorted(zip(items, item_results), key=lambda p: p[1].cost_jpy)
    names = [p[0].name for p in pairs]
    costs = [p[1].cost_jpy for p in pairs]

    min_cost, max_cost = min(costs), max(costs)
    colors = []
    for c in costs:
        if c == min_cost:
            colors.append("#2E7D32")  # 緑（最安）
        elif c == max_cost:
            colors.append("#C62828")  # 赤（最高）
        else:
            colors.append("#1f77b4")

    fig = go.Figure(
        go.Bar(
            x=costs,
            y=names,
            orientation="h",
            marker=dict(color=colors),
            text=[format_currency(c) for c in costs],
            textposition="outside",
            hovertemplate="%{y}: ¥%{x:,.0f}<extra></extra>",
        )
    )
    fig.update_layout(
        xaxis_title="原価（円）",
        height=max(300, 50 * len(names)),
        margin=dict(l=100, r=100, t=30, b=40),
    )
    st.plotly_chart(fig, use_container_width=True)


def _render_gift_summary_cards(
    gifts: list[GiftSet], gift_results: list[GiftResult]
) -> None:
    st.subheader("🎁 ギフトセット サマリ")
    if not gift_results:
        st.info("ギフトデータがありません")
        return

    pairs = list(zip(gifts, gift_results))
    max_pair = max(pairs, key=lambda p: p[1].margin_pct)
    min_pair = min(pairs, key=lambda p: p[1].margin_pct)

    col_max, col_min = st.columns(2)
    with col_max:
        st.markdown("**🥇 最大粗利**")
        st.markdown(f"**{max_pair[0].name}**")
        st.metric(
            label="粗利率",
            value=format_percentage(max_pair[1].margin_pct),
            delta=f"原価 {format_currency(max_pair[1].cost_jpy)}",
        )
    with col_min:
        st.markdown("**📉 最小粗利**")
        st.markdown(f"**{min_pair[0].name}**")
        st.metric(
            label="粗利率",
            value=format_percentage(min_pair[1].margin_pct),
            delta=f"原価 {format_currency(min_pair[1].cost_jpy)}",
        )


def render_dashboard(
    cond: ImportCondition,
    items: list[SingleItem],
    item_results: list[SingleItemResult],
    gifts: list[GiftSet],
    gift_results: list[GiftResult],
    summary: SummaryResult,
) -> None:
    _render_header(cond)
    _render_kpi_cards(item_results, summary, target_margin_pct=cond.margin_pct)
    st.divider()
    render_fx_sensitivity_chart(cond, items)
    st.divider()
    _render_item_ranking(items, item_results)
    st.divider()
    _render_gift_summary_cards(gifts, gift_results)
```
