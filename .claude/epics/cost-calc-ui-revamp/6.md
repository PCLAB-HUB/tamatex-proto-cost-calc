# Task: 為替感度チャートコンポーネントの実装

## Description

ダッシュボードで表示する「為替レート（USD）を 140〜160 の範囲で動かした時の平均原価変動」を示すインタラクティブチャートを plotly で実装する。現在の為替レートを縦線マーカーで強調し、ホバー時に正確な値を表示する。

為替は原価への影響が最も大きいパラメータの一つであり、役員・社長が最も見たい指標の一つ。

## Technical Details

### 計算ロジック

既存の計算エンジンを再利用する。`cond` の `internal_rate` を書き換えた新しい `ImportCondition` を作り、各品目の原価を再計算して平均を取る:

```python
from dataclasses import replace
from proto.engine.calc_single import calc_single_cost


def _calc_avg_cost_at_fx(
    fx: float,
    cond: ImportCondition,
    items: list[SingleItem],
) -> float:
    cond_at_fx = replace(cond, internal_rate=fx)
    costs = [calc_single_cost(item, cond_at_fx).cost_jpy for item in items]
    return sum(costs) / len(costs) if costs else 0.0
```

**注意**: `calc_single_cost` の戻り値フィールドは `proto/engine/calc_single.py` を確認して適切なものを選ぶ（`total_cost` / `cost_jpy` / `cost_per_unit` 等、実装に合わせる）。

### チャート実装

```python
import plotly.graph_objects as go
import streamlit as st
from dataclasses import replace


def render_fx_sensitivity_chart(
    cond: ImportCondition,
    items: list[SingleItem],
    *,
    fx_range: tuple[float, float] = (140.0, 160.0),
    steps: int = 11,
) -> None:
    if not items:
        st.info("品目データがありません")
        return

    fx_values = [
        fx_range[0] + i * (fx_range[1] - fx_range[0]) / (steps - 1)
        for i in range(steps)
    ]
    avg_costs = [_calc_avg_cost_at_fx(fx, cond, items) for fx in fx_values]

    fig = go.Figure()
    fig.add_trace(
        go.Scatter(
            x=fx_values,
            y=avg_costs,
            mode="lines+markers",
            name="平均原価",
            hovertemplate="USD=%{x:.1f}円: ¥%{y:,.0f}<extra></extra>",
            line=dict(color="#1f77b4", width=2),
            marker=dict(size=8),
        )
    )
    # 現在値の縦破線
    fig.add_vline(
        x=cond.internal_rate,
        line_dash="dash",
        line_color="#C62828",
        annotation_text=f"現在: USD={cond.internal_rate:.0f}",
        annotation_position="top",
    )
    fig.update_layout(
        title=f"為替感度: 平均原価 (USD={fx_range[0]:.0f}〜{fx_range[1]:.0f})",
        xaxis_title="USD (円/$)",
        yaxis_title="平均原価 (円)",
        height=400,
        hovermode="x unified",
        margin=dict(l=40, r=40, t=60, b=40),
    )
    st.plotly_chart(fig, use_container_width=True)
```

### パフォーマンス
- 11 ステップ × 6 品目 × 計算コスト ≒ 66 回の `calc_single_cost` 呼び出し → 100ms 以下で完了想定
- 大幅に遅い場合は `@st.cache_data` を使って為替入力のみでキャッシュ（将来の品目数増加に備える）
