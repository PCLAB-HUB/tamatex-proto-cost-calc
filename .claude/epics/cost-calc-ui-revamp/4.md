# Task: KPIカードコンポーネントの実装

## Description

ダッシュボードおよびシナリオ比較ビューで使用する**大きな数字表示**のKPIカードコンポーネントを実装する。`streamlit-extras` の `metric_cards` をベースに、通貨・パーセンテージ・比較差分（+/-）を見やすく表示する再利用可能な関数を提供する。

## Acceptance Criteria

### 新規ファイル
- [ ] `proto/ui/components/__init__.py` — パッケージマーカー
- [ ] `proto/ui/components/kpi_cards.py` — KPI カード描画関数
- [ ] `proto/ui/components/styles.py` — 共通 CSS 定数（色・余白）

### API
- [ ] `render_kpi_card(label: str, value: str, sublabel: str = "", delta: str | None = None, delta_color: str = "normal")` — 単独カード描画
- [ ] `render_kpi_row(cards: list[KPICardData])` — 横並びで複数カードを描画、画面幅に応じて均等割り
- [ ] `format_currency(amount: float, *, show_yen: bool = True) -> str` — 数値 → `¥1,234` 形式
- [ ] `format_percentage(value: float, *, decimals: int = 1) -> str` — 数値 → `50.6%` 形式
- [ ] `format_delta(current: float, baseline: float, *, is_currency: bool = False) -> str` — `+¥123` / `-¥456` / `+2.3%` 形式

### 機能要件
- [ ] 通貨表示は千の桁区切り（カンマ）入り
- [ ] 負の値は色で区別（赤）、正は緑（ただし粗利率「増加は良い」/ 原価「増加は悪い」の意味付けは呼び出し側で指定）
- [ ] カードに label / 主値 / sublabel（小さな補足テキスト）の3段構造
- [ ] 比較差分がある場合、カード下部に `+¥123 (前回比)` のように表示
- [ ] `streamlit-extras` の `style_metric_cards()` を呼んで全体スタイルを適用

### サンプル
- [ ] ファイル末尾 or 専用 smoke test で、3カード横並びの描画デモが確認できる（デモコメントアウト版でも可）

## Technical Details

### `proto/ui/components/kpi_cards.py` 設計

```python
"""KPIカードコンポーネント — ダッシュボードと比較ビューで共用."""
from __future__ import annotations

from dataclasses import dataclass
from typing import Literal

import streamlit as st
from streamlit_extras.metric_cards import style_metric_cards


@dataclass(frozen=True)
class KPICardData:
    label: str
    value: str
    sublabel: str = ""
    delta: str | None = None
    delta_color: Literal["normal", "inverse", "off"] = "normal"


def format_currency(amount: float, *, show_yen: bool = True) -> str:
    prefix = "¥" if show_yen else ""
    return f"{prefix}{amount:,.0f}"


def format_percentage(value: float, *, decimals: int = 1) -> str:
    return f"{value:.{decimals}f}%"


def format_delta(current: float, baseline: float, *, is_currency: bool = False) -> str:
    diff = current - baseline
    sign = "+" if diff >= 0 else ""
    if is_currency:
        return f"{sign}¥{diff:,.0f}"
    return f"{sign}{diff:.1f}%"


def render_kpi_card(
    label: str,
    value: str,
    sublabel: str = "",
    delta: str | None = None,
    delta_color: Literal["normal", "inverse", "off"] = "normal",
) -> None:
    st.metric(label=label, value=value, delta=delta, delta_color=delta_color)
    if sublabel:
        st.caption(sublabel)


def render_kpi_row(cards: list[KPICardData]) -> None:
    cols = st.columns(len(cards))
    for col, card in zip(cols, cards):
        with col:
            render_kpi_card(
                label=card.label,
                value=card.value,
                sublabel=card.sublabel,
                delta=card.delta,
                delta_color=card.delta_color,
            )
    # カード全体に streamlit-extras のスタイルを適用
    style_metric_cards(
        background_color="#FFFFFF",
        border_left_color="#1f77b4",
        border_color="#CCC",
    )
```
