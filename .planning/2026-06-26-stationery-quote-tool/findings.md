### 数式一覧（代表行 Row 9）

```
Z  = (S+T)*(1+U)+V+(W*X/1000)+Y/DM
AM = Z*AJ+(AA+AB+AC)+(AD+AE+AF)*(1+$AE$4)*$AF$4+(AG+AH+AI)*AJ+(AL/R)*AK
AO = AM*(1+AN)
AQ = AO*AP
AR = AO+AQ
BD = AS+AT+AU*AK+AV+AW+AX+AY+AZ+BA+BB+BC
BE = BD/R
BG = (AR+BE)*BF
BQ = SUM(BH:BP)
BR = AP  （関税率を引き継ぐ）
BT = BQ*(1+BR)*(1+BS)
BZ = BX*BY  （プリント型代 = 型単価×型数）
CK = (BU+BV+BW+BZ+CA+CB+CC+CD+CE+CF+CG+CH+CI+CJ)/DM
CM = CK*(1+CL)
CT = CN+CO+CP*CQ+CR+CS
CU = CT/P
DD = CW+CX+(CY+CZ+DA*DB+DC)/CV
DE = AR+BE+BG+BT+CK+CU+DD   ← 製品原価
DG = ROUNDUP(DE*(1+DF), 0)   ← 試算売価
DM = DK*DL                    ← 見積ロット
DN = DH/(1-DI-DJ)             ← 歩積込売価
DQ = DN/DP                    ← 掛率
DR = DM*DN                    ← 売上金額
DS = DH-DE                    ← 粗利額
DT = DS/DH                    ← 粗利率
DU = DM*DS                    ← 粗利金額

（第2価格体系: 償却別途）
DW = DE-CK                    ← 原価(償却除)
DY = ROUNDUP(DW*(1+DX), 0)
EE = EC*ED                     ← ロット（=DK*DL）
EF = DZ/(1-EA-EB)             ← 歩積込売価
EG = ROUNDUP(CK*(1+CL)/(1-EB), 0) ← 償却経費別途
EK = EE*EF                    ← 売上金額
EL = DZ-DW                    ← 粗利額
EM = EL/DZ                    ← 粗利率
EN = EE*EL                    ← 粗利金額
```

### R列（コンテナ積載量）の課題
- 新Excelでは数式ではなく**値が貼り付けられている**
- 旧Excelの「コンテナ積載量計算」シートにロジックあり:
  - コンテナ内寸(20FT: 2340×5000×2370mm, 40FT: 2340×11800×2370mm)
  - 製品サイズ→ダンボールサイズ→コンテナ積載ケース数→積載個数
  - ロス率10%適用
- ただし逆算してもR列の値と完全一致しない（梱包外装の膨張係数が不明）
- → プロトタイプではR列を**手動入力 or 近似計算**で対応
